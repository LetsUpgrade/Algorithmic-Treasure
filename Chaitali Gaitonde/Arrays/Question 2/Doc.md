#### Problem Statement

Given an array A (distinct elements) of size N. Rearrange the elements of array in zig-zag fashion. 

The converted array should be in form a < b > c < d > e < f. The relative order of elements is same in the output i.e you have to iterate on the original array only.

    Example 1:

    Input:
    2
    7
    4 3 7 8 6 2 1
    4
    1 4 3 2
    Output:
    3 7 4 8 2 6 1
    1 4 2 3
